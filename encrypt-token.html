編集トークンのスクランブル生成

GitHub トークンをパスワードで暗号化（スクランブル）し、config.js に貼り付ける文字列を作ります。

このページはオーナー専用です。サイトには公開・リンクしないでください。入力内容はインターネットに送信されず、すべてこのブラウザの中だけで処理されます。

使い方
  1. 上の3つの欄を入力して「スクランブル生成」を押す。
  2. 出てきた文字列をコピーする。
  3. config.js の encToken: '' の '' の中に貼り付ける。
  4. config.js を GitHub にアップロードする。
  5. 編集する人には、トークンではなくパスワードだけを伝える。
パスワードを変えたいときは、ここで作り直して encToken を差し替えればOKです。
